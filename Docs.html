Getting Started

إعداد البيئة

لتبدأ المزامنة، تحتاج أولاً لتهيئة تطبيق DevHub على جهازك الشخصي أو هاتفك.

التثبيت السريع 

curl -s https://devhub.io/install.sh | bash

ربط مستودع GitHub

يعتمد DevHub على الـ **Personal Access Token** الخاص بك لضمان السيادة الكاملة على البيانات.

1

أنشئ مستودع (Private) باسم devhub-sync.

2

أدخل الـ Token في إعدادات التطبيق.

بروتوكول التشفير

يتم اشتقاق المفاتيح عبر **Argon2id** محلياً. هذا يعني أن ملف قاعدة البيانات الخاص بك يصبح مجرد "ضجيج رقمي" غير مفهوم لأي شخص يحاول اختراقه.

DevHub

أول بروتوكول مزامنة لا مركزي مشفر للمطورين. صُمم ليمنحك السيادة الكاملة على بياناتك البرمجية عبر نفق GitHub الآمن.
