سياسة الخصوصية.

تحديث: مارس 2026 // إصدار 1.0.4

1. فلسفة "صفر بيانات"

في DevHub، نحن لا نجمع بياناتك. التطبيق يعمل كجسر مشفر بين جهازك الشخصي وحساب GitHub الخاص بك. لا تمر أي بيانات عبر خوادمنا لأننا ببساطة لا نملك خوادم وسيطة لتخزين بياناتك.

2. ما الذي يتم تخزينه محلياً؟

  • يتم تخزين مفاتيح التشفير الـ API الخاصة بك داخل "خزنة النظام" (System Keychain) المشفرة في جهازك.
  • لا يتم رفع مفتاح التشفير الأساسي (Master Key) إلى السحاب أبداً.

3. حقوق المستخدم

لك الحق الكامل في حذف كافة ملفات المزامنة من مستودع GitHub الخاص بك في أي وقت. بمجرد حذف المستودع، تنتهي علاقتنا البرمجية بالبيانات تماماً ولا تبقى لها أي نسخة احتياطية لدينا.

4. التواصل الأمني

إذا اكتشفت أي ثغرة أمنية، يرجى التواصل معنا فوراً عبر security@devhub.io. نحن نكافئ المطورين الذين يساعدوننا في تحسين أمان النظام.
